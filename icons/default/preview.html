📋 PNG変換手順

  1. 方法1: オンラインツールを使用
    CloudConvert などのツールで icon.svg をアップロードし、各サイズ（16x16, 48x48, 128x128）のPNGに変換
  2. 方法2: Node.jsスクリプトを使用
    以下のコマンドを実行:
    npm install
    npm run generate-icons
  3. 方法3: ImageMagick（CLI）
    convert -background none icon.svg -resize 16x16 icon-16x16.png
    convert -background none icon.svg -resize 48x48 icon-48x48.png
    convert -background none icon.svg -resize 128x128 icon-128x128.png
💡 ヒント: 生成されたPNGファイルは icons/ ディレクトリに配置してください。manifest.jsonで既に設定済みです。
